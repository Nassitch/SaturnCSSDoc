Installation:

To start using the CSS library, you need to install the dependency via npm:
(The library is compatible only with Angular and React.jsx).

npm install saturn-css

Then you need to run the command that executes the script and installs the absolute paths to the library:

npm run saturn-css

Basic Structure:

The library uses classes to style HTML elements. Here are some basic classes you can use:

  • Text-align: .text-center, .text-left, .text-right, .text-justify
  • Font-size: .text-1 to .text-12.
  • Line-height: .line-1 to .line-12
  • Font-weight: .font-1 to .font-9.
  • Text-transform: .text-uppercase, .text-lowercase, .text-capitalize.
  • Displays: .d-block, .d-inline, .d-none, etc.
  • Margins/paddings: .m-, .mt-, .mr-, .mb-, .ml-, .my-, .mx-, .p-, .pt-, .pr-, .pb-, .pl-, .py-, .px-.
  • Positions: .p-relative, .p-absolute, .p-fixed, .p-static, .p-sticky
  • Border: .bordered (Applies a 1px black border.)
  • Text-transform: .text-uppercase, .text-lowercase, .text-capitalize
  • Vertical-align: .vertical-middle, .vertical-baseline, .vertical-bottom, etc...
  • clamp-text: Restricts text to a maximum of 4 lines.
    Add the clamp-text class to specify the number of lines. Example for one line: clamp-text clamp-text-1

Border-radius:

To round the corners of elements, use the following classes:

  • .rounded : Applies a border-radius of 50% to create a circle.
  • .radius-none : Removes the border-radius.
  • .radius-1 to .radius-12 : Sets the border-radius to different values.

Flex:

Flex classes control the alignment and positioning of elements:

  • Horizontal Alignment: .align-center, .align-baseline, etc..
  • Vertical Alignment: .justify-center, .justify-between, etc..
  • Directions: .flex-row, .flex-column, etc..
  • Flex Wrap: .flex-wrap, .flex-nowrap, .flex-wrap-reverse
  • Gaps: .gap-1 to .gap-12.
  • Align Items: .align-start, .align-end, .align-center, etc..
  • Align Self: .align-self-start, .align-self-end, .align-self-center, etc..
  • Align Content: .content-start, .content-end, .content-center, etc..
  • Justify Content: .justify-start, .justify-end, .justify-center, etc..
  • Justify Self: .justify-self-start, .justify-self-end, .justify-self-center, etc..

Customization:

You can also customize the library by adding your own styles or replacing default values of CSS variables. Some features, such as the scrollbar or grid system, are customizable.

Usage Examples:

Here are some usage examples:

Center text, change font, and apply margins and padding:

<p class="text-center text-6 font-6 line-4 px-6 pt-2 mb-4"> Content </p>

Create a simple and responsive grid:

<div class="container">
<div class="d-flex gap-4">
<div class="col-6">Content 1</div>
<div class="col-6">Content 2</div>
</div>
</div>
